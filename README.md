# WebApp-Grade02

Single Web Application

## Dependencies
- npm ^4.0.0
- node ^7.0.0
- python ^2.6

## Installation
- Install all dependecies of backend and frontend concurrently. Only for the first time!<br>
    `npm run dev-pre-installation`

## Running
- Run backend and frontend concurrently on development.<br>
    `npm run dev`
